## Development

Install gems:

```bash
bundle install --path vendor/bundle
```

Install [overcommit](https://github.com/brigade/overcommit) hooks:

```bash
bundle exec overcommit --install
```

Run the [middleman](https://middlemanapp.com) server:

```bash
bundle exec middleman server
```
